Avalanche is a blockchain network used to record and settle transactions. xpesa uses it because payments can be fast, fees are low, and the network is reliable for small creator payments.

## What fans need to know

Fans only need to make sure their wallet is on the supported network during checkout.

## What creators need to know

Creators receive payments in their connected wallet and can withdraw available earnings to M-Pesa where supported.
