A file pack is a small bundle of files sold together. It is useful for templates, starter kits, swipe files, lesson materials, or design assets.

## Pack limits

Packs support up to 3 files. Keeping the pack small helps downloads stay clear and makes it easier for fans to understand what they are buying.

## Supported content

Use file packs for PDFs, documents, images, ZIP files, templates, or mixed learning materials.

<Steps>
  <Step title="Create a pack">
    Choose file pack mode from your dashboard.
  </Step>
  <Step title="Add files">
    Upload up to 3 files and name each item clearly.
  </Step>
  <Step title="Set the price">
    Price the pack based on the total value of the bundle, not only the number of files.
  </Step>
  <Step title="Publish the pack">
    Share the generated pack link.
  </Step>
</Steps>
