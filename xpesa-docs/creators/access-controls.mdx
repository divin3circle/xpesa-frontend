---

Access controls help you match protection to the kind of content you sell.

<AccordionGroup>
  <Accordion title="Expiry" icon="clock">
    Controls how long access lasts after payment. Use it for timed sessions, short-term resources, and offers that should not stay open forever.
  </Accordion>
  <Accordion title="One-time access" icon="rotate-ccw">
    Allows content to be opened only once. Use it for sensitive links or redemption-style offers.
  </Accordion>
  <Accordion title="Max views" icon="eye">
    Limits how many times a document can be opened. Use it when a buyer may need to return, but you still want a cap.
  </Accordion>
  <Accordion title="IP binding" icon="wifi">
    Locks access to the fan's network at payment time. Use it carefully because mobile networks can change often.
  </Accordion>
  <Accordion title="Wallet binding" icon="wallet">
    Allows only the wallet that paid to access the content. Use it for most paid documents and packs.
  </Accordion>
  <Accordion title="Watermark" icon="droplet">
    Shows the fan's wallet address on document pages. Use it when discouraging resharing matters.
  </Accordion>
</AccordionGroup>

<Note>
  Stronger controls can reduce sharing, but they may also make support cases more common. Use the lightest controls that protect the offer.
</Note>
